This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Simple Board
* Focus on implementing react functionality.
* Developed for React learning without using DBMS.

### build
* yarn install

### start
* yarn start

### CRUD
* Update
  * 수정모드 : 수정할 Row를 클릭시
  * 수정완료 : 수정된 Row를 더블클릭시
